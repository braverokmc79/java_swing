<p>&lt; 기능 &gt;</p>

<p>1. &nbsp;블로그에서 데이터를 가져와서 화면에 뿌려줌. RSS 연동</p>

<p>2. 유튜브 동영상 연동으로 &nbsp;고화질 시청 기능</p>

<p>( 32bit &nbsp;64bit 의 환경에 호환상 실행이 안될 수 있으니 jdk 설치 &nbsp;확이 &nbsp;</p>

<p>eclipse -&gt;<br />
window -&gt; preference -&gt; java -&gt; Installed JREs &nbsp;)</p>

<p>&nbsp;</p>

<p>2. &nbsp;실시간으로 기상 정보 데이터를 &nbsp;스크롤&nbsp;해서 가져와서 화면에 뿌려줌, &nbsp; ( 스크롤 &nbsp;데이터 토큰 파싱)</p>

<p>&nbsp;</p>

<p>4. 실시간 스크롤해서 가져온 &nbsp;기상 정보 데이터를 &nbsp;프린트 할 수 있는 기능.</p>

<p>&nbsp;</p>

<p>5. 백과사전 검색 기능으로 위키백과 연동으로 데이터를 가져와서 화면에 뿌려줌</p>

<p>&nbsp;</p>

<p>.6. 스케쥴러 기능 데이터 저장 , 수정, 삭제 기능 ( DB 연동 없이 데이터 입출력만으로 가능하도록 만듬 )&nbsp;</p>

<p>&nbsp;</p>

<p>7. 실시간 뉴스 데이터를 가져오는 기능</p>
